xd-数组

发表于 2018-11-05 | 更新于 2019-03-08 | 分类于 JavaSE

概述

  • 数组既可以存基本数据类型，也可以存引用数据类型

声明

一维数组

  1. 数据类型[] 数组名=new 数据类型[数组长度];
    • int[] arr=new int[3]
  2. 数据类型[] 数组名={数据1,数据2,.....}
    • int[] arr={1,3,5}
  3. 数据类型[] 数组名=new 数据类型[]{数据1,数据2,.....}
    • int[] arr=new int[]{1,3,5}

二维数组

  1. int[][] arr = new int[3][4]
    • 二维数组中有3个一维数组
    • 每个一维数组中有4个元素
  2. int[][] arr = new int[3][]
    • arr[0] = new int[2]
    • arr[1] = new int[3]
    • arr[2] = new int[4]
      • 意思就是声明时不指定一维数组元素个数，而是分别指定个数。
  3. int[][] arr = { {2,3,4},{3,6},{3,4,5,6} }

数组的toString

  • [I@38aw24c(我瞎写的)
    • [
      • 代表是一维数组（n个[就代表n维）
    • I
      • 代表是int类型
    • @
      • 固定写法
    • 38aw24c
      • 代表十六进制的地址值
