数据类型

基本数据类型

分类

整数形

  • byte
    • 1 字节 有符号的以二进制补码表示的整数
    • 默认初始化值 0
    • -128~127
  • short
    • 2 字节 有符号的以二进制补码表示的整数
    • 默认初始化值 0
    • -215~215-1
  • int
    • 4 字节 有符号的以二进制补码表示的整数
    • 默认初始化值 0
    • -231~231-1
  • long
    • 8 字节 有符号的以二进制补码表示的整数
    • 默认初始化值 0L
    • -263~263-1

浮点型

  • float
    • 4 字节 单精度 符合IEEE 754标准的浮点数
    • 默认初始化值 0.0f
    • float取值范围比long大
      • 这是通过一些规定得出的结果
  • double
    • 8 字节 双精度 符合IEEE 754标准的浮点数
    • 默认初始化值 0.0d

字符型

  • char
    • 2 字节 一个单一的 16 位 Unicode 字符
    • 默认初始化值 \u0000
    • \u0000 ~ \uffff(0~65535)
    • 注
      • char可以储存中文,
        • java是Unicode编码，char占两个字节，中文也占两个字节
      • char在内存中占两个字节，是16个bit位
      • \u0000每一个0其实代表的是16进制的0(16进制的0占四个bit位0000)，那么四个0就是代表16个bit位

布尔型

  • boolean
    • true / false
    • 默认初始化值 false

长度计算公式(n为字节数)

  • 无符号
    • 0~28n-1
  • 有符号
    • -28n-1~ 28n-1-1

转换

  • boolean类型与其他基本类型不能进行类型的转换
  • 隐式转换
    • 从低位类型到高位类型自动转换
  • 强制转换
    • 从高位类型到低位类型需要强制类型转换
    • 会损失精度
  • 常量优化机制
    • jvm编译过程中，对于默认为int的整数，当赋值给一个比int类型范围小的数据类型时(byte/short/char)，会进行判断，如果此数值没有超过被赋值的类型的范围时，jvm会自动进行一次隐式类型转换。如果超过了范围，就会报错。
    • 例
      • byte b = 127 正确
      • byte b = 128 错误

运算

  • 混合运算时byte/short/char不会互相转换，都会自动提升为int，其它类型运算时小的数据类型提升为大的.
    • byte,short,char -- int -- long -- float -- double

运算符

算数运算符

  • + - * / ++ --
  • %
    • 模
    • 左边绝对值<右边绝对值，结果为左边
    • 左边绝对值=右边绝对值，结果为0
    • 左边绝对值>右边绝对值，结果为余数
    • 结果的符号只和左边有关，与右边无关。
    • 任何一个正整数%2结果不是0就是1，可以作为切换条件。

赋值运算符

  • =
  • += -= *= /= %=

比较运算符

  • > < >= <= == !=

逻辑运算符

  • &
    • 与
    • 有false则false
  • |
    • 或
    • 有true则true
  • ^
    • 异或
    • 相同为false，不同为true
  • !
    • 非
    • 非false则true，非true则false
  • &&和&的区别(||和|同理)?
    • 最终结果一样。
    • &&具有短路效果。左边是false，右边不执行。
    • ||具有短路效果。左边是true，右边不执行。
    • &和|是无论左边是false还是true,右边都会执行

位运算符

  • &
    • 有0则0
  • |
    • 有1则1
  • ^
    • 相同则0，不同则1
    • 特点
      • 一个数对另一个数异或两次，结果为其本身。
      • a^b^b=a
  • ~
    • 按位取反
  • <<
    • 左移
    • 左边最高位丢弃，右边补齐0
    • 相当于乘以2n
  • >>
    • 右移
    • 最高位是什么，左边补齐什么。
    • 相当于除以2n
  • >>>
    • 无符号右移
    • 无论最高位是0还是1，左边补齐0

三元运算符

  • 关系表达式 ? 表达式1 : 表达式2
    • 关系表达式为boolean值
      • 真则结果为表达式1
      • 假则结果为表达式2

    • byte b = 3;
    • b = b +3;错误
      • b为变量，编译时无法判断大小是否在byte范围内。
    • b+=3;正确
    • b++;正确
      • 相当于b=(byte)(b+1)

    3. 面试用

  3. 最有效率计算2 * 8

    • 2<<3

语句

  • if
  • if else
  • for
  • while
  • do while
  • continue
    • 循环语句中使用
    • 跳出本次循环，继续下一次。
  • break
    • switch和循环语句中使用
    • 跳出循环
  • return
    • 结束方法
  • switch

    switch(表达式){
      case 值1:
       code;
       break;
      case 值2:
       code;
       break;
      …
      default:
       code;
       break;
    }
    • 表达式
      • 基本类型接收(可以自动提升为int的)
        • byte short char int
      • 引用数据类型
        • 枚举(jdk1.5)
        • String(jdk1.7)
    • case
      • 后边跟常量,不能是变量
    • break
      • 若省略，下一个case不再判断，出现case穿透现象。
      • 最后一个可以省略
    • defult
      • 可以放到任意位置，但是总是最后执行。
