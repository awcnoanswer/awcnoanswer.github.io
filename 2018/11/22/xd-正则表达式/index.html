xd-正则表达式

发表于 2018-11-22 | 更新于 2018-11-23 | 分类于 Java

概述

Pattern

  • java.util.regex.Pattern
    • 正则表达式的编译表示形式（正则对象）
    • 指定为字符串的正则表达式必须首先被编译为此类的实例
  • static Pattern compile(String regex)
    • 将给定的正则表达式编译到模式中
    • 即根据正则表达式获取正则对象
  • Matcher matcher(CharSequence input)
    • 根据正则对象获取匹配器对象
    • CharSequence
      • 是个接口，实现有String,StringBuffer....

Matcher

  • java.util.regex.Matcher
    • 匹配器对象
  • boolean matches()
    • 匹配整个字符串
    • 当且仅当整个区域序列匹配此匹配器的模式时才返回 true
  • boolean lookingAt()
    • 从字符串开头开始匹配，与matches()类似，但不需要匹配整个字符串
    • 当且仅当输入序列的前缀匹配此匹配器的模式时才返回 true
  • boolean find()
    • 查找是否有下一个匹配的序列
  • String group()
    • 获取下一个匹配的序列
    • 获取前要先查找
  • String replaceAll(String replacement)
    • 替换所有匹配序列

使用

特殊字符 （需要用\转义）

  • .
  • \
  • ^
  • $

预定义字符类

  • .
    • 任何字符。
  • \d
    • 数字：[0-9]
  • \D
    • 非数字
  • \s
    • 空白字符:[ \t\n\x0B\f\r]
  • \S
    • 非空白字符
  • \w
    • 单词字符：[a-zA-Z_0-9]
    • 字母数字下划线
  • \W
    • 非单词字符

字符类

  • [abc]
    • a、b 或 c
  • [^abc]
    • 任何字符，除了 a、b 或 c（否定）
  • [a-zA-Z]
    • [a-z[A-Z]]
    • [a-z][A-Z]
    • 并集
    • a到 z 或 A到 Z，两头的字母包括在内（范围）
  • [0-9&&[^3]
    • 交集
    • 0-9所有数字除了3

数量

  • ?
    • 0或一个
  • *
    • 0或多个
  • +
    • 一个或多个
  • {n}
    • n个
  • {n,}
    • 至少n个
  • {n,m}
    • n到m个

组

分组

  • ()
    • 从左到右从1开始编号。
      • 以 ( 顺序分组
      • 组 0 始终代表整个表达式

使用

  • \\组编号
    • 直接写组的编号即可
    • 注意转义,因为数字是字符
  • $组编号
    • 代表的是组的内容
    • 例如abccccdefgggghi将叠词用它自己替换即变成abcdefghi
      • str.replaceAll("(.)\\1+", "$1")

边界

  • ^
    • 行的开头
  • $
    • 行的结尾
  • \b
    • 单词边界
  • \B
    • 非单词边界
  • 边界有什么用？（安）
    • 比如查找三个单词的匹配项，如果有四个五个。。的也会匹配
      • abc ades aaaaa ... 匹配的结果是abc ade aaa
      • 此时加上边界就可以只找三个的

String类中的正则

  • boolean matches(String regex)

    • 判断字符串是否匹配正则
    • boolean b = str.matches(regex)相当于

  • String[] split(String regex)

    • 根据正则表达式切割字符串为字符数组
  • String replaceAll(String regex, String replacement)
    • 根据正则表达式用给定的字符串替换原字符串匹配的序列
    • str.replaceAll(regex, replacement)相当于

应用

  1. 治疗口吃

  2. ip地址排序

  3. 邮箱校验

  4. 网络爬虫
    • 一个用于获取网络中符合规则的数据的程序

JavaScript的正则

  • var regex=/^正则规则$/
    • 用//包括不用加""
    • ^n
      • 以n开头
    • n$
      • 以n结尾
