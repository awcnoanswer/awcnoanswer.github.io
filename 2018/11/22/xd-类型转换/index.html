String → ?

基本类型

  1. static int parseInt(String s)
    • 基本类型包装类中的静态parseXxx方法
    • Character没有
      • 如果可以转换，那么得到一个字符类型的数据，字符类型只能有一个字符，一个字符串有多个字符显然不正确
      • String → char
        • 先获取char[]再遍历
  2. int intValue()
    • 基本类型包装类xxxValue方法
    • 先将字符串对象封装成Integer对象
      • Integer i=new Integer("123")
      • int ii = i.intValue()

  1. Integer(String s)

  2. static Integer valueOf(String s)
