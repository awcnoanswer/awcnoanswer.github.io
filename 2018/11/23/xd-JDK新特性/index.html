xd-JDK新特性

发表于 2018-11-23 | 更新于 2018-12-06 | 分类于 Java

Jdk 5

增强for循环

interface Iterable

  • Iterator iterator()
  • 此接口为Collection的父接口
  • 实现此接口的都可以用for-each

三种迭代能否删除

  • 普通for循环
    • 可以删除,但是索引要--
  • 迭代器
    • 可以删除,但是必须使用迭代器自身的remove()方法,否则会出现并发修改异常
  • 增强for循环
    • 不能删除
    • 增强for依赖的是迭代器，会发生并发修改异常

可变参数

  • 格式
    • 修饰符 返回值类型 方法名(数据类型… 变量名){}
  • 注意事项：
    • 这里的变量其实是一个数组
    • 如果一个方法有可变参数，并且有多个参数，那么，可变参数肯定是最后一个

静态导入

  • 格式
    • import static 包名.类名.方法名
      • 静态导入导入的是类中的静态方法

枚举

Jdk 6

Jdk 7

二进制字面量

  • 用0b表示二进制
    • 0b110

数字字面量可以出现下划线

  • 运行时会去掉
    • 100_000

switch语句可以用字符串

泛型简化,菱形泛型

  • =号前写了泛型，=后只写个<>即可
    • List<String> list=new ArrayList<>();

异常的多个catch合并，每个异常用或|

标准异常处理代码

  • try-with-resources语句

jdk 8

接口中可以定义一般方法

  • 使用default关键字
    • public default void fun(){}

接口中可以定义静态方法

  • public static void fun(){}
    • 调用接口名.方法名

局部内部类访问局部变量时可以不加final

  • 但实际上还是final的
