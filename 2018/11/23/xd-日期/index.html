xd-日期

发表于 2018-11-23 | 更新于 2019-03-08 | 分类于 JavaSE

原日期组件

Date

  • java.util.Date
    • java.sql.Date是其子类

构造

  • Date()
    • 当前时间对象
  • Date(long date)
    • 1970-01-01 + date的时间对象
    • Date(0)
      • 1970-01-01 8:00:00
      • 8点是因为东八区

方法

  • long getTime()
    • 1970 年 1 月 1 日 00:00:00 到此时的毫秒值
  • void setTime(long time)
    • 将时间设置为1970-01-01 + time

abstract DateFormat

  • java.text.DateFormat
  • static DateFormat getDateInstance([int style])
    • 抽象类不能创建对象
    • 这个方法底层实际是返回了SimpleDateFormat实例
    • 相当于父类引用指向了子类对象
  • String format(Date date)
    • 将日期按指定格式变成字符串
  • Date parse(String source)
    • 字符串解析为Date

SimpleDateFormat

  • java.text.SimpleDateFormat

构造

  • SimpleDateFormat()
    • 默认格式
  • SimpleDateFormat(String pattern)
    • 指定格式

方法

  • 使用父类DateFormat的format和parse方法

abstract Calendar

  • java.util.Calendar
  • static Calendar getInstance()
    • 底层返回GregorianCalendar实例
    • 父类引用指向子类对象

常量

  • YEAR
  • MONTH
    • 0-11
  • DAY_OF_MONTH
  • DAY_OF_WEEK
    • 周日是第一天

方法

  • int get(int field)
    • 获取指定日历字段的值
  • abstract void add(int field,int amount)
    • 给指定日历字段添加或减去指定的时间量
  • void set(int field, int value)
    • 将给定的日历字段设置为给定值
  • void set(int year, int month, int date)

jdk8的日期组件

  • 新的日期API类都是final修饰而不可变的，如果遇到加减运算及格式化操作，本质上会创建新的实例。
  • 新的API操作都是线程安全的

包

  • java.time
    • 基础包，使用标准的iso-8601历法
  • java.time.chrono
    • 使用一些非标准的历法（中华民国历法(辛亥革命)，泰国佛教历法…）
  • java.time.formate
    • 包含能够格式化和解析日期时间对象的类
    • 在绝大多数情况下，我们不应该直接使用它们，因为 java.time 包中相应的类已经提供了格式化和解析的方法
  • java.time.temporal
    • 包含一些时态对象，用于找出特定时间和日期（如某月第一天）
  • java.time.zone
    • 包含支持不同时区以及相关规定的类

组件

LocalDate

  • toString()
    • 默认格式yyyy-MM-dd

获取日期实例

  • static LoaclDate now()
    • 从system clock和默认的时区获取当前时间
  • static LoaclDate now(Clock clock)
    • 从指定的clock获取当前时间
  • static LoaclDate now(ZoneId zone)
    • 从system clock和指定的时区id获取当前时间
  • static LoaclDate of(int year, int month, int dayOfMonth)
    • 从指定的year, month and day获取LocalDate实例
  • static LoaclDate of(int year, Month month, int dayOfMonth)
    • Month中是枚举
  • static LoaclDate ofEpochDay(long epochDay)
    • 就是1970-01-01加上传入的天数epochDay
  • static LoaclDate ofYearDay(int year, int dayOfYear)
    • 根据指定的year和当年的第几天获取LocalDate实例

日期变化

  • plus(long amountToAdd, TemporalUnit unit)
  • plus(TemporalAmount amountToAdd)
  • plusDays(long daysToAdd)
  • plusWeeks(long weeksToAdd)
  • plusMonths(long monthsToAdd)
  • plusYears(long yearsToAdd)
  • minus同plus

日期格式化

  • String format(DateTimeFormatter formatter)
    • 日期转成指定格式的字符串
  • static LocalDate parse(CharSequence text)
    • 将字符串转成yyyy-MM-dd格式的日期，必须是yyyy-MM-dd格式的字符串
  • static LocalDate parse(CharSequence text, DateTimeFormatter formatter)
    • 将指定格式的字符串转成yyyy-MM-dd格式的日期
    • 有内置的格式，也可以自定义
      • DateTimeFormatter.ofPattern(自定格式)

获取

  • Period until(ChronoLocalDate endDateExclusive)
    • 获取日期直接的差值
  • long until(Temporal endExclusive, TemporalUnit unit)
  • with(TemporalAdjuster adjuster)
    • TemporalAdjusters
      • static TemporalAdjuster firstDayOfMonth()
      • static TemporalAdjuster firstDayOfNextMonth()
      • static TemporalAdjuster lastDayOfMonth()
      • …
      • 有许多此类的方法返回就是TemporalAdjuster
    • 如
      • LocalDate ld = now.with(TemporalAdjusters.lastDayOfMonth())
  • with(TemporalField field, long newValue)
  • withDayOfMonth(int dayOfMonth)
    • 取本月第几天
  • withDayOfYear(int dayOfYear)
  • withMonth(int month)
  • withYear(int year)

LocalTime

  • toString()
    • 默认格式HH:mm:ss.SSS
    • 包含毫秒数
  • 基本用法同LocalDate
  • withNano(0)
    • 去除毫秒数

LocalDateTime

  • static LocalDateTime ofInstant(Instant instant, ZoneId zone)
    • 将instant按指定zone转换为LocalDateTime

Instant

  • 用在机器可读的时间格式上的，它以 Unix 时间戳的形式存储日期时间
  • 本质是一个数字时间戳，底层实际上是对System.currentTimeMillis()进行了包装

ZonedDateTime

Clock

  • 时钟提供给我们用于访问某个特定 时区的 瞬时时间、日期 和 时间的
  • Clock c1 = Clock.systemUTC()
    • 系统默认UTC时钟
    • 当前瞬时时间 System.currentTimeMillis()
  • Clock c2 = Clock.systemDefaultZone()
    • 系统默认时区时钟
  • Clock c3 = Clock.system(ZoneId.of("Asia/Shanghai"))
    • 上海时区时钟
    • 注意每次调用将返回当前瞬时时区(UTC)
  • Clock c4 = Clock.fixed(Instant.now(), ZoneId.of("Asia/Shanghai"))
    • 固定上海时区时钟

Period

  • 代表两个日期之间天，月，年的差值
  • 也可以直接使用日期中的until()完成
  • static Period between(LocalDate startDate, LocalDate endDate)
  • getDays()
  • getMonths()
  • …

Druation

  • 表示的是两个日期时间间隔的秒以及纳秒数
  • static Duration between(Temporal start, Temporal end)
  • getNano()
  • getSeconds()
  • …

旧的日期支持

  • Instant timestamp = new Date().toInstant()
    • Date to Instant
    • 然后可以把Instant转换为LocalDateTime
  • Date dt = Date.from(Instant.now())
    • Instant to Date
  • Instant time = Calendar.getInstance().toInstant()
    • Calendar to Instant
  • ZoneId defaultzone = Timezone.getDefault().toZoneId()
    • TimeZone to ZoneId
