JFrame

概述

  • JFrame 是 Frame 的一个子类，与 Frame 的使用十分相似
  • 唯一的区别在于，你不能将组件加入到JFrame中
    • JFrame 有一个 Content Pane，窗口能显示的所有组件都是添加在这个 Content Pane 中。

方法

  • getContentPane()
  • setContentPane()

JFrame添加组件有两种方式

  1. getContentPane()获得JFrame的内容面板，再对其加入组件
    • frame. getContentPane ().add(childComponent)
  2. 建立一个Jpanel或JDesktopPane之类的中间容器，把组件添加到容器中，用setContentPane()方法把该容器置为JFrame的内容面板
    • JPanel contentPane = new JPanel()
    • frame.setContentPane(contentPane)
    • 注
      • 先添加panel后设置大小，可见。。。正常显示
      • 反之，不显示，不知道是一次这样，还是就应该这样。
