xd-Html

发表于 2018-12-07 | 更新于 2019-03-08 | 分类于 UI

概述

HTML

  • Hyper Text Markup Language
  • 超文本标记语言
    • 超文本
      • 指页面可以包含非文字内容。
    • 标记
      • 使用标签将需要的内容包括起来

XHTML

  • Extensible Hyper Text Markup Language
  • 可扩展的超文本标记语言
    • 可以理解为升级版HTML

html的访问

  1. 本地
    • file:///绝对路径
  2. web访问
    • http://localhost:8080/...

小细节

单位

  • html中的数值默认单位是像素px
  • 尺寸写法
    1. 像素
      • 0px
    2. 百分百：占据副标签的百分比
      • 0%

路径

  • ./
    • 当前目录
  • ../
    • 上一级目录

注释

  • //
  • /* */
  • <!-- -->

特殊字符

  • &nbsp;空格
  • &gt;大于号
  • &lt;小于号
  • &copy;版权符号
  • &reg;注册符号

标签

基本标签

<br/>

<hr/>

<h1>~<h6>

<span>

<div>

<p>

  • 段与段之间有空行

<b>

  • 粗体

<i>

  • 斜体

<font>

属性

  • size
    • 1~7 浏览器默认为3
  • color
    1. #xxxxxx
      • 00-FF
    2. colorname
  • face
    • 字体

例

  • <font color="red" size="3" face="黑体"></font>

<img>

属性

  • src
  • alt
    • 图片无法正常显示时的文字
  • width
  • height
  • align
    • 对齐方式(top middle bottom)
    • 代表图片与相邻文本的相对位置

<a>

属性

  • href
  • name
  • target
    • _selt
      • 本页打开 默认
    • _blank
      • 空白页打开
    • 标签name
      • 在指定name位置打开

说明

  • 协议
    • 如果不指定协议，默认是file协议。(file:///)
    • 访问网上资源要指定http://
  • <a href="javascript:void(0)">
    • 取消超链接默认点击效果
  • <a href="javascript:fun()">
    • 使用javascript:前缀说明后边是js代码不是url
  • <a href="#id">
    • 锚点#

语义化标签

html5为了提高可读性提供的标签。

<header>

<footer>

列表标签

层级列表

<dl>

<dt>
  • 一个<dt>可以有多个<dd>
<dd>
  • 是对<dt>的解释

无序列表

<ul>

有序列表

<ol>

<li>
  • type
    • 1、A、a、I、i（数字、字母、罗马数字）
  • start
    • 数字，代表首项是从第几个开始。

表格标签

<table>

属性

  • border：边框
  • align：对齐方式
  • bgcolor：背景颜色
  • cellpadding
    • 单元格边框与内容的空白
  • cellspaceing
    • 单元格之间的空白
  • width：宽

说明

  • <table>的下一级标签是<tbody>不写也存在。
    • <tbody>可以有多个
    • xxx.tBodies
      • 获取所有<tbody>
    • xxx.rows
      • 获取所有行

<caption>

  • 标题

<th>

  • 用于定义表头，单元格内容默认居中，加粗。

<td>

  • colspan
    • 列合并
  • rowspan
    • 行合并
  • align
    • 对齐方式（left,right,center）

框架标签

<frameset>

属性

  • rows
    • 按行分
  • cols
    • 按列分

<frame>

  • 每一部分是一个frame
  • name
  • src
    • 加载页面的路径

注

  • 分割有两种
    1. 按具体长度划分
      • 60,*,75
        • 意思是页面分三部分60+*+75=总长度
    2. 按百分比划分
      • 25%,*
        • 页面分两部分25%和75%
  • <frameset>不能用在<body>中，通常不写<body>直接用
  • <frameset>可以嵌套
  • <frameset>已经过时，html5不支持
  • <iframe> 画中画标签

表单标签

<form>

属性

  • action
    • 提交的位置
  • method
    • 提交的方法(get/post/…)

<input>

type
  • text
    • name
    • value
      • 标签值
      • 可以省略不写，写上则为输入框默认值。
    • placeholder
      • 设置默认值
        • 类似淘宝等等网站登录时的**请输入用户名**
      • 字体为灰色，得到焦点不会消失，一输入值就消失。
  • password
    • name
    • value
      • 标签值
      • 可以省略不写，写上则为输入框默认值
  • radio
    • name
      • name一致的radio为一个组，只能单选。
    • value
      • 标签值。
      • 显示的值只是显示而已，只能代表着哪个value，与value是什么值无关。提交的是value的值。
    • checked
      • 默认选中
  • checkbox
    • name
      • name一致的checkbox为一个组，多选。
    • value
      • 标签值。
      • 显示的值只是显示而已，只能代表着哪个value，与value是什么值无关。提交的是value的值。
    • checked
      • 默认选中
  • file
    • name
  • button
    • name
    • value
      • 按钮显示名称
  • image
    • 图片按钮，也会提交（同submit）
    • name
    • src
    • alt
  • hidden
    • name
    • value
      • 标签值
  • submit
    • 不指定服务端会提交到当前页面。
    • name
    • value
      • 按钮显示名称
  • reset
    • name
    • value
      • 按钮显示名称

<select>

  • name
<option>
  • value
    • 标签值
  • selected
    • 默认选中
技巧
  • options
    • option集合
  • selectedIndex
    • 选中的option的索引
  • 获取选中的option
    • var optionElement=selectElement[selectElement.selectedIndex]
    • var optionElement=selectElement.options[selectedIndex]
  • 获取option中的值
    • var value=optionElement.value
    • var value=optionElement.innerHTML
  • 清空select中的option
    • selectElement.options.length=0

<textarea>

  • name

小结

  • 如果给服务端提交数据，必须使用表单标签，表单中组件必须有name和value。
    • text password
      • name必写
      • value省略，由用户填写
    • radio checkbox hidden
      • name和value必写
    • file
      • name必写
      • 没有value，提交的是上传的文件
    • button reset submit
      • name不写
      • value写(显示名称)

其它标签

  • <meta>
  • <link>
  • <script>
  • <abbr>
    • 缩略语

附

  • html标签的属性不要直接写，因为无法复用。用css写在style属性中
