xd-CSS

发表于 2018-12-11 | 更新于 2019-03-08 | 分类于 UI

简介

  • 层叠样式表

书写规则

  • 属性与属性之间用分号隔开
  • 属性与属性值之间用冒号隔开
  • 一个属性有多个值用空格隔开

引入方式

内嵌样式（不建议）

  • css的代码嵌入到html标签中
  • 每个html标签都有一个style属性,该属性的值就是css代码
  • <div style="属性:值;属性:值">

内部样式

  • 使用<style>标签进行css的引入
    • <style type="text/css">css代码</style>
  • 通常定义在<head>标签中,可以先加载。

外部样式

  • 将css样式抽取成一个单独css文件,用时导入。

导入方式

  1. <link>
    • html标签链接
    • <link rel="stylesheet" type="text/css" href="xxx.css"/>
  2. @import（基本不用）
    • css导入
    • <style type="text/css"> @import url("css地址"); </style>
  • link & @import方式的区别
    • link所有浏览器都支持 import部分低版本IE不支持
    • import方式是等待html加载完毕之后在加载
    • import方式不支持js的动态修改

优先级

  • 内嵌>内部>外部
  • 即后加载会覆盖先加载的。

选择器

基本选择器

元素选择器

  • html标签名{css}

id选择器

  • #id{css}

class选择器

  • .class{css}

优先级

  • id>class>元素

属性选择器

  • 基本选择器[属性='值']{css}
    • input[type="text"]{}

伪元素选择器

  • 格式
    • 标签名:伪元素
  • 例
    • <a>的伪元素选择器
      • 未访问状态 a:link{css}
      • 已访问状态 a:visited{css}
      • 悬停状态 a:hover{css}
      • 点击状态 a:active{css}

层级选择器

  • 父级选择器 子级选择器{css}

组合选择器

  • 选择器1,选择器2{css}
    • 并集

属性

文字

  • font-size
  • font-family
    • 字体

文本

  • color
  • text-docoration
    • 下划线
    • 值
      • none
      • underline
  • text-align
    • 对齐方式
    • 值
      • left
      • right
      • center(水平居中)
  • line-height
    • 设置值为height，垂直居中

背景

  • background-color
  • background-image
    • 值 格式
      • url("图片地址")
  • background-repeat
    • 平铺
    • 值
      • repeat
        • 横向纵向平铺 默认
      • no-repeat
      • repeat-x
        • 横向平铺
      • repeat-y
        • 纵向平铺

列表

  • list-style-type
    • 列表项(ul ol…)前的小标志
    • 属性值很多
  • list-style-image
    • 列表项前加图片
    • 值
      • url("图片地址")

尺寸

  • width
  • height

显示

  • display
    • 值
      • none
      • block
        • 块级显示(如前边有东西会另换一行自成一块)
      • inline
        • 行级显示(在一行显示)

盒子模型

  • 默认border padding margin都为0

border

  • border-width
  • border-color
  • border-style
  • border-top
  • border-bottom
  • border-left
  • border-right

padding

  • 边框内壁与内部元素之间的距离（内边距）

值

  • padding:1px;
    • 一个值表示上下左右都是1px
  • padding:1px 2px;
    • 两个值表示上下1px/左右2px
  • padding:1px 2px 3px;
    • 三个值表示上1px/左右2px/下3px
  • padding:1px 2px 3px 4px;
    • 四个值表示上右下左（顺时针）
  • 单独设置每个方向
    • padding-top/bottom/left/right

margin

  • 边框外壁与外部元素之间的距离（外边距）

值

  • margin:1px;
    • 一个值表示上下左右都是1px
  • margin:1px 2px;
    • 两个值表示上下1px/左右2px
  • margin:1px 2px 3px;
    • 三个值表示上1px/左右2px/下3px
  • margin:1px 2px 3px 4px;
    • 四个值表示上右下左（顺时针）
  • 单独设置每个方向
    • margin-top/bottom/left/right

布局

浮动

  • float
    • 该属性的值指出了对象是否及如何浮动
    • 值
      • left
        • 元素向左浮动。
      • right
        • 元素向右浮动。
      • none
        • 默认值。元素不浮动，并会显示在其在文本中出现的位置。
  • clear
    • 规定元素的哪一侧不允许其他浮动元素。
    • 值
      • left
        • 在左侧不允许浮动元素。
      • right
        • 在右侧不允许浮动元素。
      • both
        • 在左右两侧均不允许浮动元素。
      • none
        • 默认值。允许浮动元素出现在两侧。

说明

  • 浮动的框可以向左向右移动，直到它的边框碰到包含框或另一个浮动框的边框
  • 浮动脱离文档普通流，所以，文档的普通流中的块框表现得就像浮动框不存在一样。
  • 安
    • 如有三个div块
      • 1左浮动23没有浮动
      • 1脱离文档流，显示原本是左上开始23上下显示，1向左浮动直到碰到包含框，则会覆盖2，所以结果是13上下显示，而2被1覆盖了
    • 可以把块想象成可以流动的由固定大小形状的东西，如果三个块都浮动，一行不够用，会自动流到下一行，而如果12在第一行，3在第二行，1又比2高，想象3往第二行流动时，会卡在12交界处，而不会一直流到第二行左边。

定位

  • position
    • static
      • 默认值。没有定位，元素出现在正常的流中（忽略 top, bottom, left, right 或者 z-index 声明）。
    • relative
      • 生成相对定位的元素，相对于其正常位置进行定位。因此，”left:20” 会向元素的 LEFT 位置添加 20 像素
    • absolute
      • 生成绝对定位的元素，相对于 static 定位以外的第一个父元素进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定
      • 没有父元素，则是相对于body
    • fixed
      • 生成绝对定位的元素，相对于浏览器窗口进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
