xd-Anki

Anki

笔记类型

  • 问答
    • Basic
    • Basic (and reversed card)
      • 正面+反面显示一遍，之后，反面+正面显示一遍
    • Basic (optional reversed card)
      • =Basic+Basic (and reversed card)
      • 会出现三次
    • 注
      • 可更多次，自行添加
      • Basic (and reversed card) 和Basic (optional reversed card) 如果只在正面有内容，那么就等于Basic
  • 填空
    • Cloze

样式


.card {
 font-family: arial;
 font-size: 20px;
 text-align: center;
 color: black;
 background-color: white;
}

  • font-family: arial
    • arial为字体，可更换
  • 使用这个更改是全局性的——凡是使用这个模板的卡片都会相应更改

音频

  • [sound:名称.mp3]
    • 名称.mp3文件放在C:\Users\anThree\AppData\Roaming\Anki2\an3\collection.media
    • 其它格式应该也可以，wav我试了可以的。
